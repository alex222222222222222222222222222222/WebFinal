My casino game


This project was made in Intro to Computer Science and was actually the final project for that class where we had to pick something we wanted to make and do so using andriod studio. It took a while to make as well as made it difficult for me to code it within what time we had. I got most of the game completed however the dice game would break sadly. First you would pick a game by clicking on the button, it'll bring you to a screen that'll let you either start that game or read the rules for the game. The card game is a game where you draw a card and the computer draws a card, if you get a higher card then you win $150 otherwise you lose. The fortune wheel has you "spin" which just calls a random function and depending on whatever number you get you get a certain amount of money.
